# sondr ZMK config for Aurora Sweep

Features
[x] Bluetooth
[x] Underglow and backlighting
[x] Home row mods
[x] mac,windows,linux support
[x] Combos
[x] Esc
[x] Tab
[x] Enter
[x] Hyperroll layout
[x] Strnk layout
[x] Layer for vim usage in visual mode
[x] Symbol layer
